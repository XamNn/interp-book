# Values

Interp implements the following primitives. They can be stored into [variables](./variable.md) as well as they can be passed into [functions](./function.md) as arguments.

- [Void](./void.md)
- [Null](./null.md)
- [String](./string.md)
- [Int](./int.md)
- [Float](./float.md)
- [Bool](./bool.md)
- [List](./list.md)
- [Record](./record.md)
- [Function](./function.md)
