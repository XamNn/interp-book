# Int
Ints are 64-bit signed integers.

```cs
var int = 123;
```

## Math
The following mathematical operators are supported
```cs
1 + 2 # Addition
1 - 2 # Subtraction
1 * 2 # Multiplication
1 / 2 # Division
1 % 2 # Remainder
1 ^ 2 # Power
```

## As a type
Value ranges can be expressed as a type.
`Int[0..10]` accepts an Int that is >= 0 & < 10.
